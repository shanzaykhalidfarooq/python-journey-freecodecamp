## 📁 Project Directory

Here is a breakdown of the scripts I have built so far, ordered from my first program to my latest:

### 1. 📝 Report Card Printer
* **File:** `report_card_printer.py`
* **Concepts:** Variables, data types (`str`, `int`, `float`, `bool`), type checking with `type()` and `isinstance()`.
* **Description:** A simple script that initializes student data and verifies their data types.

### 2. 🪪 Employee Profile Generator
* **File:** `employee_profile_generator.py`
* **Concepts:** String concatenation (`+`), string slicing (`[start:end]`), and formatted f-strings.
* **Description:** Generates an employee info card and automatically extracts codes (like department and year) using index slicing.

### 3. 💵 Bill Splitter
* **File:** `bill_splitter.py`
* **Concepts:** Arithmetic operators, assignment operators (`+=`), and numeric formatting with `round()`.
* **Description:** Automatically calculates a total dining bill, adds a generous 25% tip, and splits the final cost evenly among friends.

### 4. 🎬 Movie Ticket Booking Calculator
* **File:** `movie_ticket_booking_calculator.py`
* **Concepts:** Conditional logic (`if-elif-else`), logical operators (`and`, `or`, `not`), and nested conditions.
* **Description:** A complex business-logic script that evaluates user age, membership status, showtimes, and seat choices to calculate final ticket pricing.

### 5. 🗺️ Travel & Weather Planner
* **File:** `travel_weather_planner.py`
* **Concepts:** Advanced conditional logic (`if-elif-else`), nested conditions, and boolean logical operators (`and`, `or`, `not`).
* **Description:** A decision-making script that evaluates distance, weather conditions, and transport availability to determine if a commute is possible.

### 6. 🏷️ Apply Discount Function
* **File:** `apply_discount.py`
* **Concepts:** Custom functions (`def`), input validation, return statements, and edge-case handling.
* **Description:** A robust function that calculates the final price of an item after applying a discount, complete with checks for invalid data types and out-of-range values.

### 7. 🔐 Caesar Cipher
* **File:** `caesar_cipher.py`
* **Concepts:** Helper functions, string manipulation (`str.maketrans` & `translate`), string slicing, and default arguments.
* **Description:** A classic cryptography program that encrypts or decrypts text by shifting the alphabet by a specified offset while preserving case.
